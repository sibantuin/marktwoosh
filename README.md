## ğŸš€ Getting Started

### Prerequisites

Ensure you have:

- Node.js `>=18`
- npm `>=9`
- Git

### Installation

```bash
git clone https://github.com/sibantuin/marktwoosh.git
cd marktwoosh
npm install
```

### Usage

Run in development mode:

```bash
npm run dev
```

Open `http://localhost:3000` in your browser.

To build for production:

```bash
npm run build
npm start
```
